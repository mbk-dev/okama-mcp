okama-mcp

A Model Context Protocol (MCP) server that exposes the okama investment-portfolio toolkit to AI assistants — Claude Desktop, Claude Code, Cursor, and any other MCP-compatible client.

Free and open source — no hosted service, no registration. You run it yourself: locally or on your own server.

What you can ask

With okama-mcp connected, you can ask an AI things like:

  • "Backtest a portfolio of 30% gold and 70% real estate over the last 15 years."
  • "Run a Monte Carlo retirement forecast withdrawing $1,000/month indexed to inflation over 25 years."
  • "What's the tangency portfolio of SPY, BND, and GLD with a 3% risk-free rate?"
  • "Plot the efficient frontier of SPY, BND, and GLD."

The AI calls the MCP tools, which call okama directly — no Python code needed on your side.

Install & run locally

# run straight from PyPI — no clone needed
uvx okama-mcp stdio

Requires Python ≥ 3.11. Easiest with uv; installing from source (git + Poetry) is covered in the README.

Full setup guide and client configurations (Claude Desktop, Claude Code, Cursor): see the README on GitHub.

Self-host on your server

Share one instance across your MCP clients with the streamable-http transport: 

okama-mcp http --host 127.0.0.1 --port 8765 --path /mcp

Ready-made nginx and systemd examples are in the deploy/ directory.
